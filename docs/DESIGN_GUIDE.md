# HouseBook 디자인 가이드

## 📐 전체 레이아웃 구조

### 기본 구조
- **다크 모드 우선**: 기본 테마는 다크 모드
- **사이드바 네비게이션**: 좌측 고정 사이드바 (180px)
- **메인 컨텐츠 영역**: 중앙 콘텐츠 영역 (반응형)
- **우측 패널** (선택적): 달력 화면에서 메모/상세 패널 (280px)

### 색상 팔레트

#### 다크 모드
- **배경색 (Background)**: `#1A1A1A` (거의 검정)
- **카드/패널 배경**: `#2D2D2D` 또는 `#262626`
- **사이드바 배경**: `#000000` (완전 검정)
- **텍스트 기본**: `#FFFFFF` (흰색)
- **텍스트 보조**: `#B3B3B3` (회색)
- **강조 색상 (Primary)**: `#FF4444` 또는 `#E53935` (빨간색)
- **수입 색상**: `#4CAF50` (초록색) 또는 `#2196F3` (파란색)
- **지출 색상**: `#F44336` (빨간색)

#### 라이트 모드
- **배경색**: `#FFFFFF` (흰색)
- **카드/패널 배경**: `#F5F5F5` (연한 회색)
- **사이드바 배경**: `#F8F9FA`
- **텍스트 기본**: `#212529` (진한 회색)
- **텍스트 보조**: `#6C757D` (회색)
- **강조 색상**: `#DC3545` (빨간색)

### 타이포그래피
- **앱 이름**: 볼드, 18-20px
- **페이지 타이틀**: 볼드, 24-28px
- **섹션 헤더**: 세미볼드, 16-18px
- **본문**: 14-16px
- **캡션/보조 텍스트**: 12-14px

### 레이아웃
```

│  통계 📊 │                         │  +3,500,000  │
│  예산 💰 │  일 월 화 수 목 금 토   │              │
│  리포트📄│  1  2  3  4  5  6  7   │  점심 식사:  │

│          │                         │              │
└──────────┴─────────────────────────┴──────────────┘
```

### 좌측 사이드바 (고정, 180px)
- **앱 헤더**: 로고 + 앱 이름 (빨간색 배경)
- **네비게이션 메뉴**:
  - 홈 🏠
  - 달력 📅 (현재 선택)
  - 통계 📊
  - 예산 💰
  - 리포트 📄
  - 설정 ⚙️
- 선택된 메뉴: 빨간색 배경 또는 좌측 빨간 바

### 중앙 달력 영역
- **헤더**: 
  - "달력" 타이틀
  - 월 네비게이션: `< 2024.12 > 오늘`
- **월별 요약**:
  - 수입: 파란색 (+4,500,000원)
  - 지출: 빨간색 (-2,850,000원)
  - 합계: 초록색 (+1,650,000원)
- **달력 그리드**:
  - 요일 헤더: 일~토
  - 각 날짜에 거래 내역 표시
  - 수입: 파란색/초록색, +기호
  - 지출: 빨간색, -기호
  - 선택된 날짜: 주황색/빨간색 배경

### 우측 메모 패널 (280px, 선택적)
- **제목**: "메모"
- **날짜**: 선택된 날짜 표시
- **거래 목록**: 해당 날짜의 상세 내역
  - 카테고리명: 금액
- **메모 입력**: 텍스트 영역

  - "내역" 타이틀

  - 검색 아이콘
  - 필터 아이콘
- **요약 바**:
  - 전체 건수
  - 수입/지출 합계 (화살표 아이콘)
  - 현재 잔액

  - 컬럼: 날짜, 자산, 분류, 금액, 내용

- **플로팅 액션 버튼**: 우측 하단 빨간색 원형 버튼 (+)

### 거래 추가/수정 폼
- 날짜 선택
- 유형 선택 (수입/지출/이체)
- 자산 선택 (현금/계좌/카드)
- 카테고리 선택
- 금액 입력
- 내용/메모 입력

  - "연간 보고서" 또는 "월간 보고서"
  - 연도/월 네비게이션
  - 다운로드, 인쇄, 닫기 아이콘
- **요약 카드**:
  - 수입: 파란색 카드, 위쪽 화살표
  - 지출: 빨간색 카드, 아래쪽 화살표
  - 잔액: 회색/초록색 카드
- **차트 영역**:
  - 바 차트 또는 라인 차트

  - 지출: 주황색/빨간색 바
  - Y축: 금액 (0 ~ 최대값)
  - X축: 월 (1월~12월)
- **상세 테이블**:
  - 월별 수입/지출/잔액 데이터
  - 합계 행 (굵은 글씨)
  - 수입: 파란색 텍스트
  - 지출: 빨간색 텍스트
  - 잔액: 초록색/검정 텍스트

### 기능
- 연간/월간 리포트 전환
- 전년도 비교 옵션
- PDF/Excel 다운로드
- 인쇄 기능

## 🎨 컴포넌트 디자인

### 버튼
- **Primary**: 빨간색 배경 (#FF4444), 흰색 텍스트, 8px radius
- **Secondary**: 회색 배경, 흰색 텍스트, 8px radius
- **Text**: 투명 배경, 텍스트만

### 입력 필드
- 배경: #F5F5F5 (라이트) / #2D2D2D (다크)
- 테두리: 1px solid #E0E0E0 / #404040
- 포커스: 빨간색 테두리
- 아이콘: 좌측에 아이콘 추가

### 카드
- 배경: 흰색 (라이트) / #2D2D2D (다크)
- 그림자: 약간의 그림자 효과
- 둥근 모서리: 12-16px
- 패딩: 16-24px

### 아이콘
- **식비**: 🍽️ 또는 포크/나이프 아이콘
- **교통비**: 🚌 버스 또는 🚗 자동차
- **쇼핑**: 🛒 쇼핑카트
- **카페**: ☕ 커피
- **의료**: 🏥 병원 또는 십자가
- **급여**: 💰 돈주머니 또는 달러
